## 🚀 Quick Start

</div>

<br>

```bash
# 1. Clone & enter the project
cd storyforge

# 2. Install dependencies
npm install

# 3. Configure environment (optional - for Unsplash image search)
cp .env.local.example .env.local
# Edit .env.local with your Unsplash API keys

# 4. Start the development server
npm run dev

# 5. Open your browser
open http://localhost:3000
```

## 🧰 Tech Stack Deep Dive

</div>

<br>

| Layer | Technology | Purpose |
|-------|------------|---------|
| **Framework** | Next.js 15 (App Router) | Server-side rendering, API routes, routing |
| **UI** | React 19 + TypeScript | Strictly typed component architecture |
| **Styling** | Tailwind CSS 3.4 | Utility-first, dark-mode native design system |
| **Animation** | Framer Motion 11 | Smooth panel transitions, micro-interactions |
| **Canvas** | Fabric.js 6.6 | Professional 2D canvas with object manipulation |
| **State** | Zustand 5 + Immer | Lightweight, immutable state management |
| **Storage** | LocalForage (IndexedDB) | Offline-first, persistent local data |
| **Export** | jsPDF + Sharp + JSZip | PDF generation, image processing, bulk download |
| **Fonts** | Google Fonts CDN | 40+ fonts loaded on-demand |
| **Icons** | Lucide React | Consistent, scalable iconography |
| **UX** | Sonner + Next-Themes | Toast notifications, theme switching |

## 📝 Key Features

</div>

<br>

<details>
<summary><b>🎨 Canvas & Editor</b></summary>

- **5 Aspect Ratios**: 9:16 (Stories), 1:1 (Posts), 4:5 (Feeds), 16:9 (Thumbnails), 4:3 (Standard)
- **Smart Guides**: Snap-to-center, snap-to-edges, rule-of-thirds guides visible while dragging
- **Rulers & Grid**: Professional measurement tools with toggle visibility
- **Zoom & Pan**: Mouse wheel zoom, middle-click pan, keyboard shortcuts
- **Undo/Redo**: 50-step history with command pattern implementation
- **Multi-page**: Up to 10 connected pages per story

</details>

<details>
<summary><b>📝 Text System</b></summary>

- **40+ Fonts**: Serif, Sans-serif, Display, Handwriting, Monospace families
- **Full Typography**: Size, weight, style, letter spacing, line height, alignment
- **Effects**: Opacity, background pill, underline, strikethrough, text transform
- **Inline Editing**: Double-click any text to edit directly on canvas
- **Google Fonts**: On-demand loading, no bundle bloat

</details>

<details>
<summary><b>🖼️ Image Engine</b></summary>

- **Drag & Drop**: Direct file drop onto canvas
- **URL Import**: Paste any image URL
- **Object Fit**: Cover, contain, fill modes
- **12 Filter Presets**: Chrome, Fade, Noir, Vintage, Lomo, Polaroid, etc.
- **Adjustments**: Brightness, contrast, saturation, blur, sharpness, vignette, grain, temperature, fade
- **Server-side Export**: Sharp processing for ultra-high quality output

</details>

<details>
<summary><b>🎭 Stickers & Elements</b></summary>

- **Emoji Picker**: Full Unicode emoji set
- **Shapes**: Rectangle, circle, triangle, line, arrow, star, heart, diamond
- **Decorative**: Botanical leaves, abstract blobs, minimal frames, corner brackets
- **SVG Support**: Vector elements scale infinitely

</details>

<details>
<summary><b>📤 Export & Share</b></summary>

- **Formats**: PNG, JPG, PDF (multi-page), SVG
- **Qualities**: Standard (1x), High (2x), Ultra HD (3x)
- **Scope**: Single page or all pages
- **Bulk Download**: ZIP archive for multi-page exports
- **Clipboard**: Copy rendered image directly to clipboard
- **Progress**: Real-time export progress indicator

</details>

<details>
<summary><b>💾 Storage & Privacy</b></summary>

- **100% Offline**: No data ever leaves your browser
- **IndexedDB**: LocalForage with persistent storage
- **Autosave**: Every 30 seconds + 2-second debounced saves
- **Backup/Restore**: JSON export/import for full data portability
- **Crash Recovery**: Unsaved changes detection on reload

</details>
